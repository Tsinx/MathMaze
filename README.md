# Immune Hero Math Maze (免疫英雄数学迷宫)

## 简介 (Introduction)

免疫英雄数学迷宫是一款专为4岁儿童设计的亲子协作教育游戏。通过免疫细胞与病原体战斗的趣味形式，帮助孩子学习10以内的加法运算。游戏结合迷宫探索、角色收集和简单的回合制战斗，让孩子在拯救人体的冒险中掌握基础数学技能。

Immune Hero Math Maze is a parent-child collaborative educational game designed for 4-year-olds. Through fun battles between immune cells and pathogens, it helps children learn addition within 10. The game combines maze exploration, character collection, and simple turn-based combat, allowing children to master basic math skills while adventuring to save the human body.

## 核心功能 (Core Features)

*   **迷宫探索 (Maze Exploration)**: 动态生成的迷宫地图，探索人体器官（如肺部、胃部）。
*   **角色收集 (Character Collection)**: 在迷宫中寻找并收集免疫细胞队友。
*   **数学战斗 (Math Combat)**: 遭遇病原体时进入回合制战斗，通过解答加法题目发动攻击。
*   **寓教于乐 (Edutainment)**: 结合游戏与学习，通过视觉辅助（如彩色圆点）帮助孩子理解数量关系。

## 技术栈 (Tech Stack)

*   React
*   TypeScript
*   Vite
*   Tailwind CSS
*   Zustand (State Management)
*   Lucide React (Icons)

## 开始游戏 (Getting Started)

1.  安装依赖 (Install dependencies):
    ```bash
    npm install
    ```

2.  启动开发服务器 (Start development server):
    ```bash
    npm run dev
    ```

3.  构建生产版本 (Build for production):
    ```bash
    npm run build
    ```
