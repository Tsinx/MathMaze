## 使用方法

在生成脚本中，通过以下方式指定音色：

```python
rsp = dashscope.MultiModalConversation.call(
    model='qwen3-tts-flash-2025-11-27',
    api_key=API_KEY,
    text=你的文本,
    voice='音色英文名',  # 例如: 'Serena', 'Cherry', 'Ethan' 等
    language_type='Chinese',
    stream=False
)
```
